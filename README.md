# PURPOSE

把中選會競選公報底圖裡的「格子」找出來

# DEPENDENCY

+ libpng

# INSTALL

```
$ make
```

# SYNOPSIS

```
$ ./locate_block < input.png > output.txt
```

# OUTPUT FORMAT

```
x1 y1 width1 height1
x2 y2 width2 height2
...
```
